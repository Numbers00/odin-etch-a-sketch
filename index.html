Etch-a-Sketch

This drawing app features drawing on a default 16x16 grid totaling over 420 pixels. This grid is adjustable from 1x1 to 100x100 through the green slider below. To the left of the grid are the:

Primary and Secondary Color Pickers: An HTML color picker pops up upon clicking
[P]en: Applies the selected primary color on the selected grid cell
[E]raser: Colors the selected grid cell with white
[R]GB: Each pass on the grid cell colors it with a random RGB primary color, increasing darkness each time until it goes to black

And to the right:

[G]rid: Toggles the grid on/off
[L]ock: Locks the grid dimensions
Clear: Clears the grid

Controls:

Color Pickers Switch: 0

Pen: P,1

Eraser: E,2

RGB: R,3

Grid: G,4

Lock: L,5

Clear: Can only be manually clicked*
