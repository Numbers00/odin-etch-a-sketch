Etch-a-Sketch

This drawing app features drawing on a default 16x16 grid totaling over 420 pixels. This grid is adjustable from 1x1 to 100x100 through the green slider below. To the right of the grid are the [P]en (black), [E]raser (white), [R]GB, [L]ock (locks grid size), and Clear. Each of which, save for Clear, are selectable through either their first letters or the alphanumeric [1, 2, 3, 4] respectively... Github
